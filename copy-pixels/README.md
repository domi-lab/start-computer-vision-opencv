
Author: domi-lab

Function:
- Reduce half of image size by removing a pixel between two other ones.
- Copy RGB pixels from a image to another one

Requirements:
- Linux OS
- OpenCV (tested with version 3.4.6)
- Cmake

How to run:
- Open linux terminal: Ctr + Alt + T 
- cd copy_pixels_opencv
- run: run_app.sh
